# GlobalMart Sales Data Analytics

## Project Overview
An end-to-end data analysis project aimed at optimizing sales performance and inventory management for a global e-commerce retailer. The analysis covers $1.08M in total sales and 945 total orders [5, 6].

## Key Insights
 Financial Performance: Total Profit of $328,060.57 with a 30.32% profit margin [5].
 Sales Growth: Achieved a Year-over-Year (YoY) growth of 26.74% [5, 7].
 Category Dominance: Office Supplies represent 61.25% of sales, while Technology holds the highest profit margin at 67.78% [8, 9].
 Regional Leaders: The West region is the top performer, contributing $725,348.19 in sales [6, 10].
 Customer Behavior: Average order value stands at $1,145.05 with an average of 6 items per order [5, 11].

## Technical Implementation
 Data Processing: Analyzed 50,000+ transaction records using Power Query [4].
 Optimization: Developed an inventory dashboard that reduced stockouts by 28% and overstock by 35% [4].
 SQL: Optimized queries to reduce data processing time by 40% [12].

## Strategic Recommendations
 Inventory: Focus on the "Technology" and "Office Supplies" categories which show high profitability [9].
 Discounting: Managed discount bands to ensure profit margins remain healthy, as high discounts (21-30%) significantly impacted margins (down to 10.34%) [11].
 High-Value Targets: Focus marketing efforts on the 3 identified customer cohorts responsible for 60% of revenue [4].
